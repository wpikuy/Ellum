简介：
	最近LD31要来了，我没参加过LD，于是打算自己练练手。LD限定是48小时，可惜我并没有满满两天可以拿来用，只好用时间零碎的三天来代替一下。我做游戏，是因为我享受做游戏，我有要表达的东西，我想让别人有所触动。这次我没有任何控制地把我想做的并且能做的给做了出来，这三天里我自己绘图、作曲、码代码，把零零散散的时间利用上，于是这么一个奇怪的作品就诞生了。
	这次尝试了不少新东西，最刺激的就是作曲。我以前从来没有过作曲经历，也没有看过别人作曲。在决定做横板卷轴后，我打算加点难度。以前我弄的游戏在BGM上都是直接拿的别人的FREE素材，这是我很大的遗憾。这次我无论如何要自己来，就算做得烂，做得难听，也是自己做的。我稍微熟悉了一下软件就开始编曲了。短短三天我也没拿多少时间进去，不过也算是有模有样。我最大的感受就是做出来的曲子总是和自己脑海中出现的不一样，这应该是技术问题，但是无疑我今后肯定会克服的。
	美工方面我用的剪影风格加上鲜艳的背景，强烈对比自认为还是很舒服的。本来打算手绘人物动作图可是时间太短只好用的unity的补间动画，挺遗憾的。代码没有用分发器之类的东西，都是求快所以耦合度高。
	整体来说这游戏缺点不少，但是我仍然很满意。尝试一些新东西总是让人兴奋，游戏之路真的不枯燥。

游戏方式：
←→ 移动
按住shift跑动，跑动消耗左上角橙色槽，捡到食物可以补充。
右上角蓝槽为寒冷值，在火堆旁会徐徐恢复，寒冷值降到零角色会死亡。
遇到锁孔墙壁需要捡到钥匙后通过，钥匙只能使用一次。
